# Attendance Management System

## Overview

Attendance Management System is a web-based application designed to manage student attendance efficiently. The system allows faculty members to mark attendance, monitor attendance percentages, track student records, and identify students below the required attendance threshold.

## Features

* Student registration and management
* Attendance marking (Present/Absent)
* Attendance history tracking
* Faculty dashboard
* Attendance threshold monitoring
* Attendance percentage calculation
* Undo attendance operations
* Department, Year, and Batch filtering
* Attendance analytics and reports
* Queue and Deque based attendance processing
* Hybrid sorting algorithms for student records

## Technologies Used

* HTML
* CSS
* JavaScript
* Data Structures (Queue, Deque, Arrays)
* Sorting Algorithms (Merge Sort, Insertion Sort)

# Patch-Based SVD Image Denoising

## Overview

This project implements image denoising using Singular Value Decomposition (SVD). The system removes noise from images by applying patch-based SVD techniques while preserving image quality and important visual details.

## Features

* Image denoising using SVD
* Patch-based processing
* Noise reduction analysis
* PSNR calculation
* RGB channel reconstruction
* Visualization of denoising process
* Performance evaluation metrics

## Technologies Used

* Python
* NumPy
* Matplotlib
* Pillow (PIL)
* Google Colab

## Algorithms Used

* Singular Value Decomposition (SVD)
* Patch Extraction
* Similar Patch Grouping
* Noise Thresholding
* Image Reconstruction
